# Todo list Back-end com Java Spring

## User, endpoint aberto

* `POST /users/`

    Cria usuário.

## Task, exige autenticação

* `POST /tasks/`

    Cria tarefa ligada ao usuário logado.

* `GET /tasks/`

    Lista tarefas ligadas ao usuário logado.
* `PUT /tasks/{id}`

    Atualiza tarefa ditada pelo id.
